LITE server installálása - NODE modules is létrejön
    npm install --save-dev lite-server

HAI server installálása: https://www.npmjs.com/package/hai-server
    npm install -g hai-server

-----------------------------------------------
FUTTATÁSHOZ:

1; HAI server futtatása - elindítja a hai-servert, ami a json-fájlt publikálja a megadott porton, mintha az egy adatbátis lenne.
    npm run api

2; A Lite-server futtatása
    npm start
